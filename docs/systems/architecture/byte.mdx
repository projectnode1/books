# Bit and Byte

Bit is the smallest unit of data in a computer. It can be either 0 or 1.

A byte is a common unit of data in computers. One byte consists of 8 bits.
These 8 bits can represent 256 different values because 2^8=256. For example,
if interpreted as an unsigned integer, the binary value 00100010 represents 
the decimal value 34.
